# 获取当前时间取整到整分的毫秒数
```
getCurrent() {
	let now = new Date();
	let year = now.getFullYear();      //年
	let m = now.getMonth() + 1;     //月
	let d = now.getDate();          //日
	let h = now.getHours();         //时
	let i = now.getMinutes();       //分
	let nowD = `${year}/${m}/${d} ${h}:${i}`;

	let nowDS = new Date(nowD).getTime();
	return nowDS
},

```
